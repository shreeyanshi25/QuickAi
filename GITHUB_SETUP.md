# GITHUB_SETUP.md — Make This Repo Recruiter-Ready on GitHub

A step-by-step guide to present your `QuickAi` project professionally on your GitHub profile.

---

## 1. Repo Description

When you edit the repo on GitHub, set the **Description** field to:

```
Full-stack AI Productivity Platform enabling LLM content generation, AI image rendering, and edge-native background removal. Built with Node.js, React, Groq LLaMA, and Tailwind.
```

Keep it under 350 characters. Lead with what it does, end with the tech stack keywords.

---

## 2. Add Topics / Tags

Go to your repository → click the **⚙️ gear icon** next to "About" on the right sidebar → add these topics:

```
react
nodejs
ai-tools
llama3
tailwind-css
groq
vite
expressjs
computer-vision
generative-ai
```

> ✅ Topics make your repo easily discoverable and establish the breadth of technologies you employed.

---

## 3. Pin the Repo on Your Profile

1. Go to your **GitHub profile page** (github.com/shreeyanshi25)
2. Click **"Customize your pins"** (top of the pinned section)
3. Check ✅ `QuickAi`
4. Click **Save pins**

---

## 4. Add Live Demo / Video (Optional but Highly Recommended)

Because this is a very visual Full-Stack App, recruiters want to see it work without downloading it.
- Take a screen recording (using Loom or OBS) walking through the Background Remover and Blog Generator.
- Create a GIF or upload the video directly to the `README.md` right below the header.

---
